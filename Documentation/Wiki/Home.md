# Welcome to the Elite-TaskBar Wiki!

Elite-TaskBar is a master architectural project designed to provide a classic Windows-style Taskbar replacement tool while retaining 100% of the native Windows Shell functionality.

## 🏢 Core Information
* **Company**: EliteSoftware / EliteSoftwareTech Co. 
* **Authors**: Zachary Whiteman, Susan Gemm, TheShadyRainbow4, EliteSoftwareTech Co.  
* **Target OS**: Windows 10 IoT Enterprise LTSC (Legacy compatible with Vista / 7)
* **Core Technologies**: Native Unmanaged C++ (Taskbar), .NET Framework 4.6 WinForms (Settings Applet), and Native Windows Functions.
